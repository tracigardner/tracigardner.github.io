Intended Order for the Course

The course is set up using Modules, with the idea that people will move through the course chronologically.

  • Start with the Week 01 Module, and you will have all the relevant information in one place.
  • Next move on to the Week 02 Module, and again you will have the related assignments in one place.
  • Keep going through the Modules in order, and you will always have all the related assignments and readings in one place.

The Modules are all listed on the Course Homepage.

Using Assignment Pages

If for some reason you cannot work through the modules, you may want to consult the Assignment pages. Each of the Full Drafts has its own Assignment page that includes the details on what you need to do and links to the related files.

Assignment Page Links

Each assignment page includes links to the related Criteria page, the Tip Sheet, and (if they exist) Student Examples.

  • Label Analysis Assignment
  • Technical Description Assignment
  • Instructions Assignment
  • Pitch Assignment
  • Recommendation Report Assignment

The Challenge of this Strategy

The Try-Its for each week are not listed on the Assignment pages. Go to the Modules to find the relevant Try-Its for each assignment:

  • Label Analysis Assignment: Modules 1 to 3
  • Technical Description Assignment: Modules 4 & 6
  • Instructions Assignment: Modules 5 to 6
  • Pitch Assignment: Modules 7 to 9
  • Recommendation Report Assignment: Modules 10 to 14

You can also see the connections between Assignments and Modules in the Short Course Schedule.

The Recommendation Report

The Assignment

The Recommendation Report Assignment is the best all-in-one page for the project. The page includes links to the Report Topic page, Tip Sheet, and the Criteria page.

The Assignment also has links to pages for each part of the report, with how-to details for each section.

Tip Sheet

The Tip Sheet: Recommendation Report duplicates information from the Assignment page.

It includes links to pages for each part of the Recommendation Report and points to the pages in the textbook that correlate with each section of the report.
