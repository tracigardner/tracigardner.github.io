Advice on Your Working Titles

Every working title submitted in the Week 4 Survey is listed below with some advice on whether it works as is or needs some additional details to be an effective title for a technical description.

Submitted Titles

Traci’s Advice

Particular Descriptions

  • A description of Cinnamon Toast Crunch by General Mills
  • Description of the Keurig K575 Coffee Maker
  • Functionality of Gourmia 5-Qt Digital Air Fryer
  • Technical Description of a Keurig Mini - Single Serve and Convenient
  • Understanding Your Farberware Pizza Cutter

General Descriptions

  • Description of 3D Printer
  • Description of a can opener
  • Description of A Coffee Maker
  • Description of a French Fry Cutter
  • Description of the Nature and Function of the Colander
  • Description of thermometer
  • Description of Wax Paper
  • Flatware Design and Functionality
  • In-Depth Description of a Microwave
  • Shake It Up: How a Shakercup Works
  • Technical Description of a Blender
  • Technical Description of a Cheese Grater
  • Technical Description of a Coffee Maker
  • Technical Description of a Keurig Machine
  • Technical Description of a Sauce Pan
  • Technical Description of a Stand-Mixer
  • Technical Description of a Strainer
  • Technical Description of an Air Fryer
  • Technical Description of an Immersion Blender
  • Technical Description of The Microwave
  • Technical Description: Instant Pot Pressure Cooker
  • Unscrewing the Details of the Can Opener - a Technical Description
  • What Is a Pan and How Does it Work?
  • What is a Whisk?

These titles all have a subject and scope. They all “clearly state the subject and indicate whether the description is general or particular” (Markel, p. 563).

They will all work for technical descriptions.

Clever and Clear

  • Ice Cream Scoop - Here's the scoop

As long as the reader undestands that the word scoop can mean “all the details” this title works just fine for a technical description.

Subject, but Not Scope

  • Air Fryer: A College Kid’s Best Friend
  • Microwaves: College students’ best friend
  • Refrigerator - The coldest appliance
  • The Air Fryer
  • The coffeemaker known as the Keurig
  • The Colander
  • The Ice Cream Scoop - an item that scoops ice cream
  • The inner workings of a rice cooker
  • The Metal Strainer
  • The Multi-Purpose Bottle Opener, the tool to open them all.
  • The Proper Use of a Whisk

These titles are off to a good start. They all include a subject, so the reader knows exactly what object the document is about.

The reader cannot tell however what the document will tell us about the item. Will it be instructions on how to use the object? Is it a recommendation on the best object to use? Will the document complain about the object? There’s no way for the reader to know.

Before using any of these titles, the writers need to add details that tel the reader what the document will say about the object.

Individual Titles that Need Work

Each of the working titles below has a slightly different problem, so I’ve added details for each one.

Need Details or Different Phrasing

Each of these working titles has a slightly different problem, so I’ve added details for each one.

Air Fryer Technicality
The word technicality seems off here. Usually a technicality is a specific and very technical detail. In legal TV shows and movies, people can “get off on a technicality.” That is not what the writer likely means. This title needs to more clearly indicate that the document is a technical description.
Corkscrew the cork
I’m not quite sure what the writer is going for here. I’m not sure if this title is incomplete or there’s something else going on. This title definitely needs more information to indicate the subject and scope.
Everything there is to know about Coors Light
Everything? Probably not. This title needs to be clearer about exactly what the reader is going to find out about Coors Light.
Let's Get the Cookin!
I see two possibilities here. (1) “The Cookin” is some object that I’ve never heard of that this document will talk about, OR (2) The writer meant “Let’s Get Cookin’!” Whether it is one of those two options or something else altogether, this working title needs to be revised so that the reader can tell what object the document is about and that this document is a technical description.
Sizzlin cast Iron
I’m not sure what this working title is about either. Surely not all the cast iron in existence, and I can’t find any object with that name. It seems as though more information would clear things up. For instance, Sizzlin’ Cast Iron Skillet would have let me know exactly what object the document would talk about. The title also needs some details to tell the reader that the document is a technical description.
The spatula and all its glory details
It’s clear that this working title is about a spatula. What I’m not sure about is the second part of the title: “all its glory details”? Maybe it uses the wrong word? Gloriful would make more sense, if that is what the writer means. Whatever the writer means, this title could use a clearer statement of the topic and some details that tell the reader this will be a technical description.

These Working Titles Suggest the Document Is on the Wrong Topic

Description and Instructions for an Electric Milk Frother
The assignment is to write a technical description. The Instructions are a second, separate document.
How the use of Can openers helps our cooking
The Best Uses for Tostitos
Cooking Knives and Their Many Uses Explored
These three working titles suggest that the document is about uses of the object. The assignment focuses on technical description however. These titles need to state the subject and scope more clearly so the reader knows what to expect in the document.
If you’re looking for powerADE you have come to the right place.
This working title suggests that the document will be about how to find Powerade. The writer appears to be trying for a clever title, but it doesn’t make it clear that this document will be a technical description.         
Label Analysis Report of Rao's Tomato Sauce
A Technical Analysis of a Spatula
Functional Analysis of Water Purifier
Analysis of the Kroger Fruit & Grain bar
These working titles indicate what object the document is about, but they all indicate that the document will provide an analysis. The assignment is to write a technical description. These working titles need revised to clarify what the document will tell the reader about the object.  
Technical description of an airplane
The Airplane- a vehicle that is able to leave the ground on its own power resulting in flight
These two working titles tell me that the writers are on the wrong topic altogether. The assignment is to write about an item related to food. An airplane doesn’t fit the assignment. Check the list of examples on the assignment page topics that work.
