Focus on Instructions

For the User Document assignment you’re starting this week, you are writing step-by-step instructions. It’s a genre (or kind of writing) that is often confused with process description.

The Basics

Process Description
  • A process description explains how something works.
  • If your document mostly explains background, theory, or how a system functions, you’re describing a process.
Instructions
  • Step-by-step instructions tell someone how to do something.
  • If your document guides a reader through clear actions they can complete, you’re writing instructions.

Compare the Difference

Process Description
  • Explaining how a rain barrel collects water.
Instructions
  • Showing a homeowner how to install a rain barrel.
  • Explaining how a platform routes messages.
  • Showing a user how to create an account and post.

Notices in Instructions

Notices are short, clearly labeled statements that alert readers to risks, prevent mistakes, or offer helpful guidance right before (or beside) a relevant step.

Step-by-step instructions often include notices—but each type has a specific purpose and associated color, explained below:

 Danger

 Warning

 Caution

Note

Use Danger, with a red label, when there is risk of death or severe injury.

Use Warning, with an orange label, when there is risk of harm or major damage.

Use Caution, with a yellow label, when there is possible risk.

Use Note to provide helpful tips, clarification, or troubleshooting guidance.

These are not interchangeable labels. They signal the level of risk to your reader. If everything is labeled “Warning,” then nothing stands out.
