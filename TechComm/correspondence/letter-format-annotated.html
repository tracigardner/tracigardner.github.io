The example in the table below shows the general layout and formatting instructions for letters. Note these important details:

  • The layout in the table includes more vertical space between the sections to allow for the full explanation.
  • If your letter is more than one page long, add page numbers (either at the top of each page or the bottom of each page). If you choose the top of each page, do not number the first page.
  • Skip space for your signature if you will print the letter. For this class, you can use a script font for your signature. If you are sending the letter digitally in the workplace, you will likely have a scanned version of your signature to use.
