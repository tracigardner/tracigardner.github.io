The Task

How You Do It

  1. Download the Introduction document (a word processor file).
    Note: you must download the file. Everyone in the class uses the same file, so you cannot edit the file at the link.
  2. Open the file in a word processor and add your answers, following the instructions in the document.

  3. Save the finished document once you add all your notes.
  4. Submit your document:
    • Submit your project here by 11:59 PM on Friday, April 9.
    • If you need more time, submit your document later. The assignment will close on Friday, April 30.
