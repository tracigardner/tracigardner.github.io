Since this course takes place entirely online, you have to know how to use Canvas to do well. Everything happens in Canvas: 

  • You find your readings and assignments.
  • You participate in discussions with your classmates.
  • You turn in your work and assess yourself.

If you have questions about Canvas, now is your chance to learn more about it. Even if you don’t come to this course with questions, you’re bound to learn something

The links below explain how to use the tools that are most important in this course. Follow these instructions for today’s post:

  1. Read through the entire list to get an idea of the help you can find here. You can return to this Discussion later in the term if you need help with one of these tasks.
  2. Choose at least one task that you would like to know more about. If you are unsure which to choose, I suggest you select the question on Modules, since they are a tool you will use every week.
  3. Read the details on the task you chose, and explore the related tool.
  4. If desired, write a reply that tells readers something new that you learned about Canvas.
